How to Use Pop Note AI

Basic Note Taking

  1. Select any text on a webpage.
  2. Click "Summarize" in the floating toolbar.
  3. Save the AI-generated summary as a note.

Screenshot Analysis

  1. Click the extension icon.
  2. Select "Take Screenshot" and drag to capture.
  3. Get an instant AI description of the content.

PDF Processing

  1. Open any PDF in your browser.
  2. Extension auto-detects the PDF.
  3. Click "Parse PDF" to summarize content.

Flashcard Creation

  1. Select text or save a note.
  2. Click "Create Flashcard."
  3. Review using spaced repetition.
